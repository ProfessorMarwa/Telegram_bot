# Aviator Telegram Bot

## Features
- Predicts Aviator rounds
- Supports webhook for fast response
- Runs 24/7 on Render

## Setup
1. Create a Telegram bot via [@BotFather](https://t.me/BotFather)
2. Copy the `BOT_TOKEN`
3. Deploy to Render:
   - Connect GitHub repo
   - Set environment variables
   - Deploy!
